## ⚙️ Local Setup

### 1. Clone repository

```
git clone https://github.com/YOUR_USERNAME/anand-portfolio.git
cd anand-portfolio
```

### 2. Install dependencies

```
npm install
```

### 3. Run locally

```
npm run dev
```

Open:

```
http://localhost:5173
```

---

## 🚀 Deployment (Vercel)

1. Push code to GitHub
2. Go to https://vercel.com
3. Import repository
4. Click **Deploy**
